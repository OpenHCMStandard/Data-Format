Definition Type: SimpleType
Name: globalId
Containing Schema: hcm-pdt.xsd
Documentation: 
A global identifier is unique within the file, not necessarily across files. It is composed of upper case and lower case letters as well as numbers. It shall not contain any whitespace.
    Different parts may be separated by colon.
It is based on the XML Schema simple type
    xs:token.

    Remark:The reason why the XML Schema type
    xs:IDis not used for this purpose is that
    xs:IDis bound to many restrictions with respect to the characters that are allowed to be used. Using
    xs:tokenas basic type allows for more flexibility with respect to the use of Unicode characters and white spaces.
The uniqueness property of an attribute or element of type
    globalIdcan be achieved by means of so called identity constraints, which is a feature of the XML Schema specification.
